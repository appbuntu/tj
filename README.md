# Theme Juice

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'theme-juice-cli'
```

And then execute:
```
$ bundle
```

Or install it yourself as:
```
gem install theme-juice-cli
```

## Usage

#### Help
```
tj
```

#### Create theme
```
tj create [<theme-name>]
```

#### Delete theme
```
tj delete <theme-name>
```
